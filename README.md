# 🏎️ MZ Tune: Web App

Welcome to **MZ Tune**, a dedicated web platform designed to centralize and enhance the automotive customization experience, connecting car enthusiasts with top-tier tuning brands.

## Project Overview
**MZ Tune** serves as a digital distributor that bridges the gap between official tuning partners and the end-user. By consolidating multiple brands and their respective catalogs into a single web interface, we provide customers with an effortless way to discover parts and performance modifications for their vehicles.

### Key Features
* **Centralized Distribution:** Direct access to products from our official project partners.
* **Unified Marketplace:** A single platform to compare and browse different tuning styles and components.
* **User-Centric Design:** Simplified and minimalistic navigation to help users visualize and select the best upgrades for their cars.

## Tech Stack
This project is built using modern frameworks to ensure high performance and scalability:

* **Frontend:** Angular
* **Backend:** Node.js / Angular
* **Database:** Sql Developer

## 🚀 Getting Started
The project is currently in the development phase and is not yet hosted online. To run the application locally, follow the steps below using **Visual Studio Code**.

### Prerequisites
* **Node.js**
* **Angular CLI** installed globally (`npm install -g @angular/cli`)
* **SQL Developer**

### Execution (Git)

1.  **Launch the Backend:**
    ```bash
    cd backend
    npm install
    npm run dev
    ```

2.  **Launch the Frontend:**
    ```bash
    cd frontend
    npm install
    ng serve --o
    ```
